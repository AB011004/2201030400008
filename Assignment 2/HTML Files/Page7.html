Muscle Building Workout

Warm-up (5 minutes):

  • Light cardio (e.g., jogging, jumping jacks)
  • Dynamic stretches (e.g., arm circles, leg swings)

Main Workout (45-60 minutes):

Note: Rest for 30-60 seconds between sets and 1-2 minutes between exercises.

Muscle Group Exercise Sets Reps
Chest Push-ups (modification: incline push-ups, decline push-ups, knee push-ups) 3 8-12
Back Rows (dumbbells, bodyweight, cable machine) 3 8-12
Legs Squats (modify with bodyweight or weights) 3 10-15
Shoulders Overhead press (dumbbells, barbell) 3 8-12
Core Plank 3 30-60 seconds hold

Cool-down (5-10 minutes):

  • Light cardio (e.g., walking, slow jogging)
  • Static stretches (e.g., hamstring stretch, quad stretch, chest stretch)
